Track A

업무 개선

My Work Improvement

👤 개인 👥 팀

현업에서 마주하는 다양한 업무를 Kiro로 자동화하여, 일하는 방식의 질적인 변화를 만드는 것을 목표로 합니다.

구현 예시

  • 정산 업무 자동화
    메일 데이터 취합 → 사내 양식 수정 → 결재 자동 처리
  • 보고서 생성 효율화
    지점별 매출 데이터 수집/가공 → 리포트 자동 생성
  • 시장 동향 모니터링
    키워드 기반 뉴스 수집 → 요약 보고서 자동 발송
Track B

신규 서비스 개발

New Service Development

👤 개인 👥 팀

IGAW의 제품 경쟁력을 높이거나, 데이터 경쟁력을 높이는 등 상용화 가능성이 있는 아이디어를 MVP 형태로 구현하는 것을 목표로 합니다.

구현 예시

  • B2B 솔루션
    문서 자동 분류·분석으로 리스크를 도출하는 AI 서비스
  • 신규 서비스 기능
    AI 기반 개인화 추천 엔진 또는 맞춤형 데이터 분석 모듈
  • SaaS 아이템
    소상공인/프리랜서 대상 유료 구독형 AI 툴
